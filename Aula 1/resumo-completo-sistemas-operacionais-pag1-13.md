# 1. O Que é um Sistema Operacional?

Um sistema operacional (SO) é o software que atua como intermediário
entre hardware, aplicações e usuários.

Ele organiza, controla e abstrai os recursos do sistema computacional.

------------------------------------------------------------------------

## 1.1 SO como Máquina Estendida

O hardware é complexo. Programar diretamente envolve:

-   Registradores
-   Interrupções
-   Endereços físicos
-   Controladores de dispositivos

O SO cria abstrações:

-   Arquivos
-   Processos
-   Memória virtual

Essas abstrações tornam o ambiente utilizável.

------------------------------------------------------------------------

## 1.2 SO como Gerenciador de Recursos

Recursos limitados:

-   CPU
-   Memória
-   Disco
-   Entrada/Saída

O SO decide:

-   Quem usa
-   Quando usa
-   Por quanto tempo
-   Com qual prioridade

Objetivos:

-   Eficiência
-   Justiça
-   Segurança

# 3. Evolução Histórica

## Primeira Geração (1945--1955)

-   Sem sistema operacional
-   Programação em máquina
-   Operação manual

## Segunda Geração (1955--1965)

-   Sistemas em lote
-   Cartões perfurados
-   Monitor residente

## Terceira Geração (1965--1980)

-   Multiprogramação
-   Time-sharing
-   Conceito de processo

## Quarta Geração (1980--Presente)

-   PCs
-   Interfaces gráficas
-   Redes
-   Sistemas distribuídos

------------------------------------------------------------------------

# 4. Conceitos Fundamentais

## Processo

Programa em execução com estado próprio.

## Memória

Gerenciamento e proteção.

## Arquivos

Abstração de armazenamento.

## Entrada/Saída

Comunicação com dispositivos.

## Chamadas de Sistema

Interface entre aplicação e SO.
