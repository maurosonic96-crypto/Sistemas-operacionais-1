# 1. O que é um Sistema Operacional?

O sistema operacional (SO) é o software que atua como intermediário
entre o hardware e os programas de aplicação.

Ele possui duas funções centrais:

## 1.1 SO como Máquina Estendida

O hardware é complexo (registradores, interrupções, controladores,
etc.).\
O sistema operacional cria **abstrações** que tornam o uso do computador
mais simples.

Exemplos:

-   Arquivos → abstraem discos físicos\
-   Processos → abstraem execução de programas\
-   Memória virtual → abstrai memória física

Hardware complexo → organização lógica.

------------------------------------------------------------------------

## 1.2 SO como Gerenciador de Recursos

O computador possui recursos limitados:

-   CPU\
-   Memória\
-   Armazenamento\
-   Dispositivos de entrada e saída

O sistema operacional decide:

-   Quem usa cada recurso\
-   Por quanto tempo\
-   Em que ordem\
-   Com qual prioridade

Objetivos: eficiência, justiça e controle.

------------------------------------------------------------------------

# 2. Estrutura Básica de um Sistema Computacional

Camadas de um sistema:

1.  Hardware\
2.  Sistema Operacional\
3.  Programas de Aplicação\
4.  Usuário

O SO fornece serviços por meio de chamadas de sistema.

------------------------------------------------------------------------

# 3. Evolução Histórica

## 3.1 Primeira geração (1945--1955)

-   Sem sistemas operacionais\
-   Programação direta em máquina\
-   Operação manual

## 3.2 Segunda geração (1955--1965)

-   Sistemas em lote (batch)\
-   Cartões perfurados\
-   Monitor residente

## 3.3 Terceira geração (1965--1980)

-   Multiprogramação\
-   Time-sharing\
-   Vários usuários simultâneos

## 3.4 Quarta geração (1980--presente)

-   Computadores pessoais\
-   Interfaces gráficas\
-   Redes\
-   Sistemas distribuídos

------------------------------------------------------------------------

# 4. Conceitos Fundamentais

## Processo

Programa em execução com estado próprio.

## Memória

Gerenciamento e proteção entre programas.

## Arquivos

Abstração para armazenamento persistente.

## Entrada/Saída

Comunicação com dispositivos externos.

## Chamadas de sistema

Interface entre programas e o sistema operacional.

------------------------------------------------------------------------

# 5. Ideia Central

O sistema operacional existe para:

-   Esconder a complexidade do hardware\
-   Organizar o compartilhamento de recursos\
-   Garantir proteção entre processos\
-   Fornecer ambiente estável

Essência:

O sistema operacional transforma caos físico em ordem lógica.
